# work-reporter

工作汇报写作助手插件，包含 OKR 周报和月度绩效两个 skill。

## Skills

### monthly-performance-writer

把每周周报内容提取、聚合、升华成月度绩效自评。

**触发方式**：
- "帮我写月度绩效"
- "写个月度自评"
- "把这几周周报总结成月度绩效"

**功能**：
- 自动从周报提取并聚合工作事项
- 经验深度分析，体现思考和方法论沉淀
- 不足积极美化，展示进取心
- 自动记录领导反馈，下月自动关联"落实情况"

### okr-weekly-writer

把 git commit log 或工作记录改成 OKR 对齐的周报。

**触发方式**：
- "帮我写周报"
- "这周干了啥"
- "帮我整理一下这周做了什么"

**功能**：
- 自动扫描 git commit log（支持多项目、多日期范围）
- 按用户 OKR 文件归类对齐
- 支持手动模式（不扫 git，直接输入内容）
- 配置和 OKR 持久化，老用户一步到位

## 安装

将本目录添加为 Claude Code 或 Codex 的插件即可。
